# Phase 1 - Ingestion

Ce répertoire contient le code pour l'acquisition et la préparation des données.

## Structure des étapes

Les sous-répertoires suivent le format: `<numéro>_<nom_étape>/`

- `01_DataAcquisition/` - Connexion aux sources de données
- `02_Parsing/` - Nettoyage et parsing des données brutes
- `03_Chunking/` - Découpage des documents
- `04_Embedding/` - Génération des vecteurs sémantiques

## À implémenter

Les pipelines d'ingestion seront implémentés selon les besoins du projet.
